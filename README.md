# 🍱 Local Food Wastage Management System

## Project Overview
A system to reduce food wastage by connecting food providers (restaurants, grocery stores) with receivers (NGOs, individuals) in need.

## Tech Stack
- Python
- SQL (SQLite)
- Streamlit
- Data Analysis (Pandas)

## Dataset
- `providers_data.csv` - Food providers details
- `receivers_data.csv` - Receivers details
- `food_listings_data.csv` - Available food listings
- `claims_data.csv` - Food claims data

## Features
- ✅ View all 4 datasets
- ✅ 13 SQL Queries with results
- ✅ Filter food by city, type, meal type
- ✅ CRUD Operations (Add, Update, Delete)
- ✅ Data Visualizations

## How to Run
1. Clone this repository
2. Install requirements: `pip install streamlit pandas`
3. Run app: `streamlit run app.py`

## Live App
Click here to view the app - https://food-wastage-management-mmawg5s29nyyhcigah4afr.streamlit.app/

## SQL Queries Covered
1. Providers & Receivers per City
2. Provider Type Contributions
3. Provider Contact Info by City
4. Top Receivers by Claims
5. Total Food Available
6. City wise Food Listings
7. Most Common Food Types
8. Claims per Food Item
9. Top Providers by Successful Claims
10. Claim Status Percentage
11. Average Quantity per Receiver
12. Most Claimed Meal Type
13. Total Donation per Provider
